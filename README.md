# מנתח אודיו מתקדם 🎵

אפליקציית Python גרפית מתקדמת לניתוח קבצי אודיו עם ממשק בעברית מבוסס PySide6.

## תכונות עיקריות

### 📊 ניתוח מוזיקלי
- **BPM (Beats Per Minute)** - זיהוי אוטומטי של קצב השיר
- **מפתח מוזיקלי** - זיהוי המפתח (C, D, E, וכו')
- **סולם** - זיהוי סולם (Major/Minor)
- **ספירת פעימות** - מספר הפעימות בשיר
- **ניתוח ספקטרלי** - Spectral Centroid, RMS Energy, Zero Crossing Rate

### 🎵 נגן אודיו מובנה
- נגינה, השהיה ועצירה
- סרגל התקדמות עם בקרה ידנית
- בקרת עוצמה
- תצוגת זמן נוכחי/כולל

### 💾 ייצוא תוצאות
- ייצוא כל התוצאות לקובץ Markdown (.md)
- פורמט מסודר וקריא
- כולל תאריך ושעה של הניתוח

### 📁 מידע על הקובץ
- שם הקובץ ונתיב מלא
- גודל הקובץ
- משך השיר (HH:MM:SS)
- תדר דגימה (Sample Rate)
- ערוצים (Mono/Stereo)
- עומק סיביות (Bit Depth)

### 🎤 מטא-דאטה
- אמן
- כותרת השיר
- אלבום
- ז'אנר

## בניה לקובץ EXE עצמאי

### הוראות בניה עם PyInstaller

```bash
pyinstaller --onefile --windowed --icon=icon.ico --add-data "icon.ico;." audio_analyzer.py
```

**הסבר הפרמטרים:**
- `--onefile` - יוצר קובץ exe אחד עצמאי
- `--windowed` - מסתיר את חלון הקונסול (GUI בלבד)
- `--icon=icon.ico` - מגדיר את האייקון של קובץ ה-exe (מה שרואים ב-Explorer)
- `--add-data "icon.ico;."` - מוסיף את האייקון לתוך ה-exe כדי שיוצג בחלון ובשורת המשימות

**חשוב:** ודא שקובץ `icon.ico` נמצא באותה תיקייה כמו `audio_analyzer.py` לפני הבניה.

### מה ההבדל?
- **`--icon`** משפיע רק על האייקון של קובץ ה-exe עצמו (בExplorer)
- **`--add-data`** דואג שהאייקון יהיה זמין בזמן ריצה להצגה בחלון ובשורת המשימות
- הקוד משתמש ב-`resource_path()` כדי למצוא את האייקון גם במצב פיתוח וגם אחרי אריזה

הקובץ המוכן יימצא בתיקייה `dist/audio_analyzer.exe`

## איך משתמשים?

1. **הפעל את האפליקציה**
   ```bash
   python audio_analyzer.py
   ```

2. **בחר קובץ אודיו**
   - לחץ על כפתור "בחר קובץ אודיו"
   - בחר קובץ מהמחשב שלך

3. **נתח את הקובץ**
   - לחץ על כפתור "נתח קובץ"
   - המתן לסיום הניתוח (יוצג בלוג)

4. **צפה בתוצאות**
   - כל הנתונים יוצגו בממשק המסודר:
     * מידע על הקובץ (שמאל)
     * ניתוח מוזיקלי (מרכז)
     * מטא-דאטה (למטה)

## טכנולוגיות

- **PySide6** - ממשק גרפי מודרני
- **Librosa** - ספריית ניתוח אודיו מתקדמת
- **NumPy** - חישובים מספריים
- **SoundFile** - קריאת מידע על קבצי אודיו
- **Mutagen** - קריאת מטא-דאטה

## פרטים טכניים

### BPM Detection
האפליקציה משתמשת באלגוריתם `librosa.beat.beat_track()` לזיהוי הקצב.

### Key Detection
זיהוי המפתח המוזיקלי מבוסס על ניתוח Chromagram (CQT) וזיהוי הצליל הדומיננטי.

### ניתוח ספקטרלי
- **Spectral Centroid** - "מרכז הכובד" של הספקטרום, מעיד על "בהירות" הצליל
- **RMS Energy** - עוצמת האנרגיה הממוצעת בדציבלים
- **Zero Crossing Rate** - קצב חציית האפס, מעיד על תכולת התדרים הגבוהים
